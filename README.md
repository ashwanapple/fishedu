# Fish Education Game

A small browser game where players catch fish swimming across the screen.

Built with **Phaser** and **TypeScript**.

---

# Tech Stack

* Phaser (game framework)
* TypeScript
* Vite (dev server + bundler)
* Node.js
* Git

---

# Requirements

Install the following before running the project:

* Node.js (version 18+ recommended)
* npm (comes with Node)
* Git

---

# Setup (First Time)

Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/fish-catching-game.git
cd fish-catching-game
```

Install dependencies:

```
npm install
```

---

# Running the Game

Start the development server:

```
npm run dev
```

Then open the URL printed in the terminal (usually):

```
http://localhost:5173
```
---

# Project Structure

```
src/
  scenes/        game scenes (home, gameplay, catalogue)
  objects/       fish, player cursor, etc
assets/          images and game assets
index.html       entry point
```
